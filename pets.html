Responsive Behavior Explanation

When the screen size is reduced, the second row (with the pet names) breaks first because the `col-md` class allows for more flexible, fluid columns that adjust to the size of the screen. The first row (with the letters) will remain intact longer as the `col-sm-2` class defines a fixed column width for smaller screens.
